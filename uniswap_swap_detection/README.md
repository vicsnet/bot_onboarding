# uniswap swap detection

## Description

<p> This bot detect swap that occurs on uniswap v3 pool </P>

## Supported chain

<li> Polygon </li>

## Alerts

<li>It is fired when swap is detected</li>
<li> it emmits the token address and the pool address </li>
<li> Finding type is an info </li>
